## Built for the Hedera Hello Future Apex Hackathon

ClawPay is the payment layer for AI agents on Hedera. We enable any MCP-compatible AI agent to autonomously pay for tool calls using the x402 protocol with native HTS USDC on Hedera.

## Key Features

- **Native Hedera integration** — HTS USDC payments, HCS audit trails, sub-second finality
- **Open-source SDK + CLI** — `@clawpay-hedera/sdk` on npm
- **16 Hedera tools** — gasless write operations, smart analytics, basic reads
- **Server marketplace** — auto-indexed registry with tool discovery
- **No-code monetization** — wrap any free MCP server with a payment proxy
- **vlayer web proofs** — cryptographic proof of API responses (integration ready for v2.0)
- **ChatGPT Apps SDK** — compatible with OpenAI's Connectors for widget rendering
