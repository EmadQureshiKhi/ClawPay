# Examples

This directory contains small, self-contained examples demonstrating how to use ClawPay components and services.

- Each subfolder is its own package (added to pnpm-workspace) so you can `pnpm install` and `pnpm dev` within it.
- Keep examples minimal and focused on one concept.

Current examples:

- `hedera-tools/` – Example MCP server with paid tools on Hedera (HTS USDC payments via x402)
- `chatgpt-apps-sdk-nextjs-starter/` – OpenAI ChatGPT Apps SDK integration
- `vlayer-client-example/` – VLayer web proofs client example
